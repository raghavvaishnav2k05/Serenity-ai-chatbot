🌿 Serenity – Mind Mingle AI

Serenity is an AI-powered mental wellness companion designed to provide a safe, supportive, and accessible space for users to monitor their emotional well-being. Built with Python and Streamlit, the application combines mood tracking, empathetic conversations, and wellness resources to encourage self-reflection and promote mental health awareness.

✨ Key Features

🧠 Smart Mood Tracking

Track daily emotional states through an interactive mood slider and receive personalized feedback based on current feelings.

💬 Empathetic AI Conversations

Engage in supportive and meaningful conversations through an intelligent chat interface. The system uses sentiment-aware, keyword-driven responses and is designed for future integration with Large Language Models (LLMs).

🌬️ Wellness & Mindfulness Support

Access curated mental wellness resources, including breathing exercises, mindfulness techniques, stress-management tips, and professional support information.

🎨 Calming User Experience

Features a clean and soothing interface built with custom CSS, ensuring an engaging and comfortable user experience.

⚡ Real-Time Interaction

Simulates natural AI response behavior with realistic thinking delays, creating a more human-centered interaction experience.

🚀 Tech Stack

* Frontend & Backend: Streamlit
* Programming Language: Python 3.x
* UI Styling: Custom CSS
* Future Enhancements: Gemini/OpenAI API Integration, Mood Analytics Dashboard, Personalized Wellness Recommendations

🎯 Project Objective

Serenity aims to bridge the gap between technology and mental wellness by offering users a judgment-free environment to express emotions, track mental health patterns, and access immediate self-care resources. The platform demonstrates how AI can be leveraged to create meaningful, socially impactful applications that prioritize user well-being.

🛠️ Installation

git clone https://github.com/your-username/serenity-mind-mingle.git
cd serenity-mind-mingle
python -m venv .venv
# Activate environment
# Windows
.venv\Scripts\activate
# Linux/Mac
source .venv/bin/activate
pip install -r requirements.txt
streamlit run chatbotSerenity.py
