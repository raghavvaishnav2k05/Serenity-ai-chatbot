🌿 Serenity - Mind Mingle AI

Serenity is a lightweight, empathetic mental health companion designed to provide a safe digital space for users to express their feelings, track their daily moods, and access immediate wellness resources.

Built using Streamlit and Python, this project focuses on clean UI/UX and immediate accessibility for mental health support.

✨ Features

🧠 Intelligent Mood Check-in: A sidebar-integrated slider that allows users to log their current state with real-time feedback.
💬 Empathetic Chat Interface: A responsive chat system that uses keyword-based logic (designed for future LLM integration) to provide supportive responses.
🎨 Custom UI/UX: Styled with custom CSS for a calming, professional aesthetic.
📚 Quick Resources: Instant links to breathing exercises, mindfulness tips, and professional mental health assistance.
⏳ Realistic Interaction: Simulated "thinking" delays to make the AI interaction feel more natural.
🚀 Tech Stack

Frontend/Backend: Streamlit
Language: Python 3.x
Styling: Custom CSS via Markdown Injection
